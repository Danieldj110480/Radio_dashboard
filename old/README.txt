Paquete del dashboard de Radio UTPL exportado por Bax.

Archivos principales:
- dashboard.html -> dashboard principal actual
- index.php -> entrada del sitio para servir el dashboard
- .htaccess -> reglas de cache/redirección
- radio-planner-state.json -> estado vivo del planner/dashboard

Archivos de referencia / históricos:
- radio-planner-v1.html
- radio-planner-v2.html
- radio-utpl-dashboard-visual.html
- radio-utpl-dashboard.md
- radio-utpl-tracker.md

Si vas a editar manualmente, normalmente lo más importante es:
1) dashboard.html
2) radio-planner-state.json
3) index.php
4) .htaccess
